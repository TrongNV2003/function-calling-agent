# Agento-v1
- Self-built Function calling AI Agent using model Qwen2.5-72B-Instruct

## Usage
### Install requirement
```sh
pip install requirements.txt
```

### Execution
For run demo:
```python
python main.py
```

Run with UI:
```python
python -m streamlit run function_calling_agents/interface/gui.py
```
